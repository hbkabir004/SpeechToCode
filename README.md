## How to run code

1. Clone the repository
   ```sh
   git clone https://github.com/learnwithsumit/voice-to-code.git
   ```
2. Install Dev Dependencies
   ```sh
   yarn
   ```
3. Run the 'index.html' file in Google Chromer Browser
