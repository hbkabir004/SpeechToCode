## How to run code

1. Clone the repository
   ```sh
   git clone https://github.com/hbkabir004/SpeechToCode.git
   ```
2. Install Dev Dependencies
   ```sh
   npm install --global yarn
   ```
   ```sh
   yarn
   ```
3. Run the 'index.html' file in any Chromium Browser (Google Chrome, Brave).
